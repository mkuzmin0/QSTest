How to use it:

- Button "+" is used to add new nodes. Just select a node you want to be a parent, enter new node name in the "Selected Node Value" text input control(it could be confusing:)) and click "+" button.

- Button "-" is used to delete nodes(with the whole subtree). Just select a node you want to delete and click "-" button.

- Button "a" is used to edit node values. Select a node you want to edit, enter new value in the "Selected Node Value" text input control and click "a" button.

- Button "Apply" is to save all changes to the DB(emulation).

- Button "Reset" brings application to the initial state.

Also, different types of nodes are marked with colors:

New nodes

Edited nodes

Deleted nodes
